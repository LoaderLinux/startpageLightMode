This is a simple startpage featuring [Nazrin](https://en.touhouwiki.net/wiki/Nazrin) from the [Touhou Project](https://en.touhouwiki.net/wiki/Touhou_Wiki).

It is written in pure HTML & CSS without JavaScript.
